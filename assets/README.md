# 오프라인 에셋 폴더

여행 전에 아래 파일들을 채워 넣으세요. 빌드 시 서비스 워커가 전부 프리캐싱하므로
**빌드/배포 후 최초 1회 접속**만 하면 비행기 모드에서도 전부 열립니다.

| 경로 | 내용 |
|---|---|
| `vouchers/disneyland_qr.png` | 디즈니랜드 티켓 QR 코드 (클룩 바우처 PDF에서 추출) |

- 장소 지도는 별도 이미지 없이 각 카드의 "주소 복사" 버튼으로 주소를 복사해 지도 앱에 붙여넣는 방식으로 대체했습니다.
- 호텔/항공권/eSIM 등 예약 정보는 이미지 캡처 대신 `archive.voucherDetails`에 텍스트로 정리해 접이식 카드로 보여줍니다.
- 포맷은 `webp` 권장 (용량 절약). 파일명은 `src/data/travelData.js`의 경로와 일치해야 합니다.
- 새 바우처 이미지(QR 등)를 추가하면 `voucherDetails` 항목의 `qrImage`에 경로를 넣고 여기에 파일을 두면 끝.
- 일요일(귀국일) 오전 일정은 아직 미정 — `travelData.js`의 `undecided`에서 정리 중. 정해지면 해당 항목을 `timelines`의 4일차로 옮기세요.
